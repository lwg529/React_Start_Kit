### react start
react+react-router+mobx+webpack项目脚手架

### 启动介绍

`
git clone ******.git
cd React_Start
yarn 
yarn start
`
打开浏览器127.0.0.1:3000即可

### 目录介绍

conifg: 项目的配置，包括环境，路径，代理和webpack的配置
script: build打包脚本，start本地启动脚本
scr: 
  api是请求的url，
  component是组件，
  module是mobx的数据管理，
  page是页面的入口
  util: 公共函数的配置
  index.js 项目的入口
  theme.less 主题配置
